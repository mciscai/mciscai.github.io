ALGO-119 寂寞的数

试题 算法训练 寂寞的数

问题描述

道德经曰：一生二，二生三，三生万物。

对于任意正整数 nnn，我们定义 d(n)d(n)d(n) 的值为 nnn 加上组成 nnn 的各个数字的和。例如，d(23)=23+2+3=28d(23)=23+2+3=28d(23)=23+2+3=28，d(1481)=1481+1+4+8+1=1495d(1481)=1481+1+4+8+1=1495d(1481)=1481+1+4+8+1=1495。

因此，给定了任意一个 nnn 作为起点，你可以构造如下一个递增序列：
n,d(n),d(d(n)),d(d(d(n))),⋯n,d(n),d(d(n)),d(d(d(n))),\cdotsn,d(n),d(d(n)),d(d(d(n))),⋯

例如，从 333333 开始的递增序列为：33,39,51,57,69,84,96,111,114,120,123,129,141,⋯33, 39, 51, 57, 69, 84, 96, 111, 114, 120, 123, 129, 141, \cdots33,39,51,57,69,84,96,111,114,120,123,129,141,⋯

我们把 nnn 叫做 d(n)d(n)d(n) 的生成元，在上面的数列中，333333 是 393939 的生成元，393939 是 515151 的生成元，等等。有一些数字甚至可以有两个生成元，比如 101101101，可以由 919191 和 100100100 生成。但也有一些数字没有任何生成元，如 424242。我们把这样的数字称为寂寞的数字。

输入格式

一行，一个正整数 nnn。
