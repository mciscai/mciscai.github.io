BASIC-15 字符串对比

试题 基础练习 字符串对比

问题描述

给定两个仅由大写字母或小写字母组成的字符串（长度介于 111 到 101010 之间），它们之间的关系是以下 444 中情况之一：

  1. 两个字符串长度不等。比如 BeijingBeijingBeijing 和 HebeiHebeiHebei。

  2. 两个字符串不仅长度相等，而且相应位置上的字符完全一致（区分大小写），比如 BeijingBeijingBeijing 和 BeijingBeijingBeijing。

  3. 两个字符串长度相等，相应位置上的字符仅在不区分大小写的前提下才能达到完全一致（也就是说，它并不满足情况 222）。比如 beijingbeijingbeijing 和 BEIjingBEIjingBEIjing。

  4. 两个字符串长度相等，但是即使是不区分大小写也不能使这两个字符串一致。比如 BeijingBeijingBeijing 和 NanjingNanjingNanjing。

编程判断输入的两个字符串之间的关系属于这四类中的哪一类，给出所属的类的编号。

输入格式

包括两行，每行都是一个字符串。

输出格式

仅有一个数字，表明这两个字符串的关系编号。

输出格式

按从小到大的顺序输出满足条件的四位十进制数。

思路

回文数就是从前往后和从后往前都是同一个数字。

由于题目要求输出的数字只有四位数，个位和千位相同且十位和百位相同。我们有两种方法处理：

  1. 我们遍历所有的四位数，当某个四位数满足要求，就输出。
  2. 由于个位和千位相同且十位和百位相同，由于千位的数字可以是 1∼91\sim 91∼9，百位可以的数字是 0∼90\sim 90∼9，我们可以构造这些回文数，然后输出。
